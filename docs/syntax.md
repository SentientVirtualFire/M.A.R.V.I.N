## data types
---------------------------------
*  variables should be allowed to be either defined with datatype in beginning or loosely typed.

ints:
```
int a=1 
a = 1 
```

floats:
```
flt a = 1.0
a = 1.0
```

strings:
```
str a = "ten"
a = "ten"
```

lists:
```
lst a = [1,0]
a = [1,0]
```

dicts:
```
dct a = {1:0}
a = {1:0}
```

Line objects have their own section
